# OmdbClient

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.2.4.

## Deploy

* Node & npm
* Angular: `npm install -g @angular/cli`
* Clone repo: 
* Install dependencies:
  * `cd omdd-client`
  * `npm install`
* Deploy local server: `npm start`
